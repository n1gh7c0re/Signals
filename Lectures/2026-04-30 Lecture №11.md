## 1. Введение: от двоичных кодов к недвоичным

Ранее мы изучали **двоичные коды БЧХ**, где информация кодируется битами (0/1). Однако реальные каналы часто используют **недвоичную модуляцию**. 
 Кроме того, ошибки в канале часто группируются в **пакеты**.

**Коды Рида–Соломона (РС)** — это недвоичные коды, являющиеся частным случаем БЧХ над полем $GF(q)$ с $q=2^m$.  
Их ключевые преимущества:
- один символ (например, байт) может содержать несколько битовых ошибок, но для кода это **одна** символьная ошибка;
- **максимальное минимальное расстояние** для заданных параметров (достигают границы Синглтона);
- высокая эффективность при борьбе с пакетами ошибок.

---

## 2. Математическая основа

### 2.1. Поле Галуа и длина кода

Код РС строится над полем $GF(q)$, где $q = p^m$ (чаще $p=2$).  
**Длина кода** максимальна: $n = q - 1$.  
> Ненулевые элементы поля образуют циклическую группу порядка $q-1$, и мы сопоставляем каждому элементу $\alpha^i$ свою позицию в кодовом слове (нулевой элемент не используется).

### 2.2. Порождающий полином

В кодах БЧХ порождающий полином $g(x)$ — это НОК минимальных многочленов последовательных степеней примитивного элемента $\alpha$.  
Для кодов РС мы выбираем **минимальные многочлены степени 1**, поскольку корни лежат в том же поле $GF(q)$.

Общий вид:

$$
g(x) = (x - \alpha^{b})(x - \alpha^{b+1}) \cdots (x - \alpha^{b+d-2}),
$$

где:
- $\alpha$ — примитивный элемент $GF(q)$;
- $b$ — целое (обычно $b=1$);
- $d$ — конструктивное (минимальное) расстояние кода.

**Следствие:** степень $g(x)$ равна $d-1$, следовательно $n - k = d-1$, откуда $k = n - d + 1$.

### 2.3. Граница Синглтона и MDS-свойство

Для любого линейного кода $(n,k,d)$ справедливо:

$$
d \le n - k + 1.
$$

Коды РС **достигают** этого неравенства: $d = n - k + 1$. Такие коды называются **максимально разделимыми** (MDS).

## 3. Пример построения кода над $GF(2^4)$

### 3.1. Поле $GF(16)$

Примитивный полином: $p(x) = 1 + x + x^4$.  
Примитивный элемент $\alpha$ удовлетворяет $\alpha^4 = 1 + \alpha$.  
Ниже приведены элементы поля:

| Степень $\alpha^k$ | Полином (коэф. при $1,x,x^2,x^3$) | Двоичный вектор |
|--------------------|------------------------------------|------------------|
| 0                  | 1                                  | 0001             |
| 1                  | $\alpha$                           | 0010             |
| 2                  | $\alpha^2$                         | 0100             |
| 3                  | $\alpha^3$                         | 1000             |
| 4                  | $1+\alpha$                         | 0011             |
| 5                  | $\alpha+\alpha^2$                  | 0110             |
| 6                  | $\alpha^2+\alpha^3$                | 1100             |
| 7                  | $1+\alpha+\alpha^3$                | 1011             |
| 8                  | $1+\alpha^2$                       | 0101             |
| 9                  | $\alpha+\alpha^3$                  | 1010             |
| 10                 | $1+\alpha+\alpha^2$                | 0111             |
| 11                 | $\alpha+\alpha^2+\alpha^3$         | 1110             |
| 12                 | $1+\alpha+\alpha^2+\alpha^3$       | 1111             |
| 13                 | $1+\alpha^2+\alpha^3$              | 1101             |
| 14                 | $1+\alpha^3$                       | 1001             |

### 3.2. Код $(15,11)$ с $d=5$ (исправление 2 ошибок)

Положим $b=1$, $d=5$. Корни $g(x)$: $\alpha^1, \alpha^2, \alpha^3, \alpha^4$.  

$$
g(x) = (x+\alpha)(x+\alpha^2)(x+\alpha^3)(x+\alpha^4) \quad (\text{в поле характеристики 2 } x-\alpha = x+\alpha).
$$

Степень $g(x)=4$, $k = n-4 = 11$.


### 3.3. Код $(15,9)$ с $d=7$ (исправление 3 ошибок)

$n=15$, $d=7$ ⇒ $t=3$, $k = 15-6 = 9$.  
Порождающий полином:

$$
g(x) = (x+\alpha)(x+\alpha^2)(x+\alpha^3)(x+\alpha^4)(x+\alpha^5)(x+\alpha^6).
$$

Степень 6. Например, коэффициент при $x^5$ равен сумме всех корней, которая после упрощений даёт $\alpha^{10}$.

## 4. Алгоритм декодирования Питерсона–Горнстейна–Зирлера (PGZ)

Этот алгоритм работает для кодов РС (и для двоичных БЧХ). Далее описан для случая $t$ исправляемых ошибок (фактическое число ошибок $\nu \le t$).

### 4.1. Синдромы

Пусть передано кодовое слово $c(x) = c_0 + c_1x + \dots + c_{n-1}x^{n-1}$,  
принято $r(x) = c(x) + e(x)$, где $e(x)$ — полином ошибок.

Так как $c(\alpha^j)=0$ для $j = b, b+1, \dots, b+d-2$ (корни g(x)), то синдромы:

$$
S_j = r(\alpha^{b+j-1}) = e(\alpha^{b+j-1}), \qquad j = 1,2,\dots, d-1.
$$

При $b=1$: $S_j = r(\alpha^j)$ для $j = 1\dots d-1$.

### 4.2. Представление через локаторы и значения ошибок

Пусть произошло $\nu$ ошибок ($\nu \le t$).  
- Позиции ошибок: $i_1, i_2, \dots, i_\nu$ (различны).  
- **Локаторы ошибок:** $X_h = \alpha^{i_h}$ (ненулевые элементы поля).  
- **Значения ошибок:** $Y_h = e_{i_h}$ (элементы поля).

Тогда:

$$
S_j = \sum_{h=1}^{\nu} Y_h X_h^{\,j}, \qquad j = 1,2,\dots, d-1.
$$

### 4.3. Полином локатора ошибок

Введём полином, корнями которого являются **обратные** локаторы:

$$
\Lambda(x) = \prod_{h=1}^{\nu} (1 - X_h x) = 1 + \lambda_1 x + \lambda_2 x^2 + \dots + \lambda_\nu x^\nu.
$$

Тогда $\Lambda(X_h^{-1}) = 0$ для всех $h$.

### 4.4. Линейная система для коэффициентов $\lambda_i$

Умножая (1) на $\lambda_{\nu}, \lambda_{\nu-1}, \dots$ и суммируя, получаем для $j = \nu+1, \dots, 2\nu$:

$$
\lambda_1 S_{j-1} + \lambda_2 S_{j-2} + \dots + \lambda_\nu S_{j-\nu} = -S_j.
$$

В матричной форме (при $\nu = t$):

Коэффициенты $\lambda_i$ находятся путем решения системы линейных уравнений вида:

$$
\begin{cases}
\lambda_1 S_1 + \lambda_2 S_2 + \cdots + \lambda_t S_t = -S_{t+1}\\
\lambda_1 S_2 + \lambda_2 S_3 + \cdots + \lambda_t S_{t+1} = -S_{t+2}\\
\vdots\\
\lambda_1 S_t + \lambda_2 S_{t+1} + \cdots + \lambda_t S_{2t-1} = -S_{2t}
\end{cases}
$$


### 4.5. Этапы декодирования (PGZ)

1. **Вычислить синдромы** $S_1, S_2, \dots, S_{2t}$ (где $t = \lfloor (d-1)/2 \rfloor$).

2. **Определить фактическое число ошибок** $\nu$
   
4. **Составить систему (3)** для $\nu$ неизвестных $\lambda_1 \dots \lambda_\nu$ и решить её (например, методом Гаусса над $GF(q)$).

5. **Построить полином локатора** $\Lambda(x) = 1 + \lambda_1 x + \dots + \lambda_\nu x^\nu$.

6. **Найти корни $\Lambda(x)$** перебором всех ненулевых элементов поля. Если $\Lambda(\beta)=0$, то локатор $X = \beta^{-1}$. Позиция ошибки $i = \log_\alpha X$.

7. **Вычислить значения ошибок $Y_h$** — решить систему:
   
$$
\sum_{h=1}^{\nu} Y_h X_h^{\,j} = S_j, \quad j = 1,\dots,\nu.
$$
   
9. **Исправить принятое слово:** $c_{i_h} = r_{i_h} - Y_h$ (вычитание в поле GF(q)).

## 5. Визуальная блок-схема алгоритма PGZ

```
┌─────────────────────┐
│  Принятый вектор r  │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│  Вычисление         │
│  S1, S2, ..., S2t   │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│  Решение системы    │
│  (3) → λ₁...λ_ν     │
└──────────┬──────────┘
           ▼
┌──────────────────────┐
│  Λ(x)=1+λ₁x+...+λ_νxⁿ│
└──────────┬───────────┘
           ▼
┌─────────────────────┐
│  Поиск корней Λ(x)  │
│  (перебор по GF(q)) │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│  Локаторы X_h       │
│  Позиции ошибок i_h │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│  Решение системы    │
│  для значений Y_h   │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│  Коррекция:         │
│  r - e → codeword   │
└─────────────────────┘
```

## 6. Сравнительная таблица: двоичный БЧХ vs код Рида–Соломона

| Характеристика            | Двоичный БЧХ                          | Код Рида–Соломона                     |
|---------------------------|---------------------------------------|---------------------------------------|
| **Алфавит**               | $\{0,1\}$                             | Элементы $GF(2^m)$ (например, байты)  |
| **Длина**                 | $n = 2^m - 1$ (макс.)                 | $n = 2^m - 1$                         |
| **Избыточность**          | $n-k = m \cdot t$ (обычно)            | $n-k = 2t$                            |
| **Минимальное расстояние**| $d \ge 2t+1$ (может быть больше)      | $d = 2t+1$ (точно)                    |
| **Достижение границы Синглтона** | редко                           | **всегда** (MDS-код)                  |
| **Борьба с пакетами ошибок** | ограничена                         | отличная (один символ = пакет битов)  |
| **Пример**                | $(15,7)$ с $d=5$, $t=2$               | $(15,11)$ с $d=5$, $t=2$              |
